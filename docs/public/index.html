Seatbelt

The Go framework that scales with your business

Seatbelt is a Go web framework used at Mart.ai that allows us to:

  • Develop new features as quickly as possible,
  • Manage existing features as the grow, and
  • Maintain successful features long term.

We think Seatbelt is pretty great, and we hope you will too. Check back in a month or two for the official launch.
